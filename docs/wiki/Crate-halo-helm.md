# Historical: 1bit-helm

`1bit-helm` was the earlier native desktop pane plan. The current primary UI/control surface is AMD GAIA, pointed at:

```text
http://127.0.0.1:13306/api/v1
```

Open WebUI is the secondary UI on:

```text
http://127.0.0.1:3000
```

Do not document `1bit-helm` as the current primary client unless that project is revived and verified against the active proxy stack.
