# 9070 XT bench — Bonsai-1.7B-IQ1_S on `gfx1201` (RDNA 4)

_Captured 2026-04-28 against `lemond` on `:13305` (Lemonade Server, llama.cpp `b8668` Vulkan lane) on a Ryzen 7 9800X3D + Radeon RX 9070 XT box (`ryzen.bcloud.tailnet`, CachyOS rolling, kernel 7.0.0-rc7-3). Single-shot OpenAI-compat curls; no warmup excluded._

## Box

- **CPU:** AMD Ryzen 7 9800X3D (8-core, 16-thread)
- **dGPU:** AMD Radeon RX 9070 XT (Navi 48, **`gfx1201`**, RDNA 4) — 16 GB GDDR6, ~640 GB/s
- **iGPU:** Granite Ridge integrated graphics (`gfx1036`, secondary, not used for this bench)
- **OS:** CachyOS rolling, kernel `7.0.0-rc7-3-cachyos-rc`
- **Stack:** `lemonade-server 10.2.0` (AUR) + `llamacpp:vulkan b8668`
- **Backend chosen by Lemonade `auto`:** `vulkan` (consistent with our earlier finding that mainline IQ1_S compute shaders > stock ROCm kernels on RDNA)
- **NPU:** none (dGPU box; install.sh's `has_xdna_npu()` autodetect correctly skipped FLM/XRT)

## Side-by-side vs strixhalo iGPU (8060S, `gfx1151`)

Same model (`lilyanatia/Bonsai-1.7B-requantized:Bonsai-1.7B-IQ1_S.gguf`), same Lemonade endpoint, same prompts.

| Decode @ output length | strixhalo 8060S iGPU | ryzen 9070 XT dGPU | Δ |
|---|---:|---:|---:|
| short (~30 tok) | 281 | 287 | +2 % |
| 100w (200 tok) | 267 | **303** | +13 % |
| 250w (350 tok) | 261 | **394** | **+51 %** |
| 500w (683 tok) | 255 | **413** | **+62 %** |

| Prefill @ prompt length | strixhalo 8060S iGPU | ryzen 9070 XT dGPU | Δ |
|---|---:|---:|---:|
| short (16 tok) | 1235 | 1403 | +14 % |

| long (~30 tok, 500w) | 2220 | **4166** | +88 % |

## Headlines

- **Decode** holds **~280 → 413 tok/s** on the 9070 XT across the sweep, scaling *up* with longer output as the bandwidth advantage compounds.
- **Prefill** scales **72 → 4166 tok/s** as prompt grows — nearly 2× the strixhalo iGPU's prefill ceiling on long prompts.
- **Steady-state decode advantage** of the discrete RDNA 4 card over the Strix Halo APU: **+51-62 %** at realistic essay-length outputs. Short-output decode is bandwidth-noise-limited and barely differs.
- **The win is most pronounced where it matters** — long-form generation (essays, code, multi-paragraph reasoning). For short replies (chat, tool calls), the iGPU is essentially the same speed, so the dGPU box pays its power cost without much return on those workloads.

## Reproduce

```sh
# on the 9070 XT box
git clone https://github.com/bong-water-water-bong/1bit-systems
cd 1bit-systems
./install.sh                      # autodetects no NPU, skips FLM lane
lemonade backends install llamacpp:vulkan
lemonade pull lilyanatia/Bonsai-1.7B-requantized:Bonsai-1.7B-IQ1_S.gguf

M=user.Bonsai-1.7B-requantized-Bonsai-1.7B-IQ1_S.gguf
curl -s http://127.0.0.1:13305/v1/chat/completions \
  -H 'Content-Type: application/json' \
  -d "$(jq -nc --arg m "$M" --arg p "Write a 250 word essay on local AI inference." \
    '{model:$m,messages:[{role:"user",content:$p}],max_tokens:350}')" \
  | jq -r '.timings | "decode: \(.predicted_per_second | floor) tps"'
```
