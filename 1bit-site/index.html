What it is

A serving stack built around 1.58-bit ternary weights. Hand-written HIP kernels target gfx1151; everything above — OpenAI-compatible HTTP, MCP bridge, sessions, sampler — is Rust. A 70-billion-parameter model fits in 128 GB of unified memory at 1.58 bits per weight, which is the whole reason this runs on a mini-PC. Weights and prompts never leave the machine.

Install

Three paths. Pick one.

AppImage — one file, any modern Linux distro:

curl -fsSLO https://github.com/bong-water-water-bong/1bit-systems/releases/download/v0.1.0/1bit-systems-0.1.0-x86_64.AppImage
echo "8a964f89bdef68ed914c04fcf23092ac642c424bb70f74dc10e8558e93b94036  1bit-systems-0.1.0-x86_64.AppImage" | sha256sum -c -
chmod +x 1bit-systems-0.1.0-x86_64.AppImage && ln -sfn "$PWD/1bit-systems-0.1.0-x86_64.AppImage" ~/.local/bin/1bit

AUR — Arch, Manjaro, EndeavourOS, CachyOS:

yay -S 1bit-systems-bin

Source — developer path, bootstraps ROCm against gfx1151:

curl -fsSL https://1bit.systems/install.sh | sh
