The APU play

Unified CPU + iGPU + NPU on one 128 GB LPDDR5 pool at 256 GB/s. No PCIe tax, no host-device copy, no discrete-card bottleneck. The whole box is the accelerator, not one tile of it. That's the wedge — and we treat it that way. See the aggregator plan.

iGPU — shipping

Radeon 8060S

gfx1151, RDNA 3.5. Native HIP ternary GEMV running 126 GB/s / 49% of LPDDR5 peak on the live Bonsai kernel. Our kernels, no hipBLAS.

NPU — evaluating

XDNA 2 · 50 TOPS

ONNX Runtime + VitisAI Execution Provider (AMD official) is the primary lane. STX-H Linux path validated in principle, still pending shim. Target: prefill offload at L ≥ 33.

CPU — aggregating

16× Zen5 cores

AVX2 ternary GEMM session-kernel measured 7261 tok/s-equivalent per matmul @ 6912×2048. Full router integration in review, not PR-gated yet.

Install

curl -fsSL https://1bit.systems/install.sh | bash
  • First token in ~60 seconds on a primed Strix Halo box.
  • Runs unprivileged. No sudo password leaves the machine; the script prompts locally.
  • Idempotent — rerun anytime to repair or upgrade. Recompiles machine-specific kernels.

Prefer git? Clone the monorepo and run ./install.sh directly. Full install options in docs/install.

How it fits together

The shape of the system, top to bottom. Rust above the HIP line, C++ below. Nothing in between.

          +---------------------------------------+
          |   helm (desktop) / voice PWA / CLI    |    Rust egui · PWA · 1bit CLI
          +----------------------+----------------+
                                 |
          +----------------------v----------------+
          |   1bit-server (axum, OpenAI-compat)   |    /v1/chat/completions · SSE · Bearer
          +----------------------+----------------+
                                 |
          +----------------------v----------------+
          |   1bit-router  (CPU · iGPU · NPU)    |    phase-aware dispatch over LPDDR5
          +---+------------------+---------------++
              |                  |                |
   +----------v----+   +---------v--------+   +---v-----------+
   |  CPU lane     |   |  iGPU lane       |   |  NPU lane     |
   |  AVX2 ternary |   |  HIP GEMV @ 49%  |   |  ORT+VitisAI  |
   |  (aggregating)|   |  of LPDDR5 peak  |   |  (evaluating) |
   +----------+----+   +---------+--------+   +---+-----------+
              |                  |                |
          +---v------------------v----------------v+
          |   128 GB LPDDR5-8000 @ 256 GB/s        |    unified, cache-coherent, zero-copy
          +----------------------------------------+

The moat is the shared memory pool, not any one tile. iGPU decodes, CPU aggregates prefill, NPU handles bulk matmul at L ≥ 33 when the path lights up.

The 1bit family

Fourteen Rust crates, one monorepo, one job each. C++ HIP kernels live below the FFI line in rocm-cpp/. Everything runs as a user-systemd unit.

1bit-server

OpenAI-compat HTTP. /v1/chat/completions with SSE. Bearer-gated via Caddy. 2.4 MB binary.

1bit-router

Backend dispatcher: iGPU / CPU / NPU. Default is HIP; aggregator splits prefill when CPU lane lights up.

1bit-hip

FFI into rocm-cpp HIP kernels. 49%-of-LPDDR5-peak ternary GEMV lives on the other side.

1bit-core

Pure parsers. mmap, zero I/O beyond the file handle. Only crate with zero deps beyond std.

1bit-voice

Sentence-boundary streaming TTS. 1.23 s first audio. 17 unit tests.

1bit-echo

WebSocket voice server, Opus 20 ms frames. Cancellation path on disconnect. 11 tests.

1bit-whisper

STT via whisper.cpp on gfx1151. Streaming partials on roadmap.

1bit-kokoro

TTS, 24 kHz mono WAV, onnxruntime. af_sky / am_michael / bf_emma / bm_george.

1bit-mcp

Stdio MCP bridge. 19 tools: 17 specialists + skill_manage + memory_manage.

1bit-agents

17 typed specialists. Discord + GitHub watchers keep the fleet online.

1bit-cli

1bit status · doctor · install · skill · memory · npu · burnin · power. One command, whole stack.

1bit-landing

LAN dashboard on :8190. Live metrics, /_live/stats SSE.

1bit-lemonade

OpenAI-compat gateway on :8200. Hermes + AMD GAIA interop.

1bit-helm

Native Rust desktop (egui). Embeds inference in-process or talks to systemd on :8180. No Electron.

1bit-mlx

Apple Silicon backend, feature-gated. Not shipped by default; kept for parity research.
