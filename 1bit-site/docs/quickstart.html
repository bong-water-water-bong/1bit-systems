Quickstart

First token in 60 seconds on a primed box. Longer if the install script is still compiling kernels (it is, the first time — up to 4 hours).

One-shot: install + test

curl -fsSL https://1bit.systems/install.sh | bash
source ~/.profile
1bit status

# get the bearer the installer printed (or read it from the Caddyfile)
BEARER="$(cat ~/.config/1bit systems/bearer)"

curl -s -H "authorization: bearer $BEARER" \
  http://localhost:8180/v1/chat/completions \
  -H "content-type: application/json" \
  -d '{"model":"halo-1bit-2b","messages":[{"role":"user","content":"one sentence, please"}],"stream":false}'

Streaming with SSE

curl -N -H "authorization: bearer $BEARER" \
  http://localhost:8180/v1/chat/completions \
  -H "content-type: application/json" \
  -d '{"model":"halo-1bit-2b","messages":[{"role":"user","content":"count to ten, slowly"}],"stream":true}'

Expected: a stream of data: {...} lines terminated by data: [DONE]. This is the OpenAI shape — any compat client works.

Pick the model

Models live in ~/.cache/1bit systems/models/. The installer seeds three:

  • halo-1bit-2b — our baseline. 2B params, TQ1 pack, default.
  • bitnet-b1.58-2B-4T — Microsoft's reference ternary, TQ1 pack.
  • bonsai-1.7b-tq2 — PrismML ternary Bonsai. Fastest per param.

1bit list models shows what's installed. See Models to add your own.
