Voice

Four crates make up the voice lane. Worst-case today: ~1.5 s mouth-to-ear. Sentence-boundary streaming shaves 30–50% when the LLM speaks more than a fragment.

TTS (kokoro)

POST /v1/audio/speech
authorization: bearer <token>
content-type: application/json

{
  "model": "kokoro",
  "input": "one sentence of speech, please.",
  "voice": "af_sky",
  "format": "wav"
}

Voices: af_sky, am_michael, bf_emma, bm_george. Output: 24 kHz mono WAV.

STT (whisper)

POST /v1/audio/transcriptions
authorization: bearer <token>
content-type: multipart/form-data

file=<audio.wav>
model=whisper-small.en
response_format=verbose_json

Returns segments + timestamps. Streaming partials are on the roadmap — see Whisper streaming plan.

Echo (duplex WS)

WebSocket at wss://<your-box>/echo. Opus 20 ms frames both directions. Binary frames are audio; text frames are control/status. Cancellation path on client disconnect cancels the in-flight LLM generation.

The PWA

Mobile-first UI at /voice/. Add to home screen; first-visit dialog picks voice + personality + name. Press-and-hold the cyan orb to talk. See mobile onboard for the setup flow.
