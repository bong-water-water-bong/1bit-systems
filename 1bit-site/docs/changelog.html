Changelog

Canonical source: GitHub releases. Key cutovers below.

2026-04-21 — site overhaul

  • Calm technical voice locked on this site; GH README gets the personality.
  • Docs section cut clean into install / quickstart / architecture / models / api / voice / desktop / troubleshooting / faq / contributing.
  • New hero: "Local AI. Strix Halo. Bare metal." + 6-tile proof band.
  • Bench numbers refreshed to 2026-04-20 snapshot.

2026-04-20 — APU-aggregator pivot

  • Strategy change: positioning shifts from "ternary runtime for the iGPU" to "APU aggregator for CPU + iGPU + NPU."
  • CPU AVX2 session kernel measured 7261 tok/s-equivalent @ 6912×2048 — CPU lane is viable as a prefill surface.
  • Lemonade 10.2 shipped Linux NPU support; VitisAI EP route locked in per Rule E.

2026-04-19 — attention rewrite + RoPE fix

  • Split-KV Flash-Decoding attention kernel landed; 6.78× @ L=2048, bit-exact.
  • RoPE convention fix: interleaved → HF split-half. PPL 524 → ~12 on wikitext; 4.29 → 1.04 on repetition text.
  • KV-cache reset bug fix after ~200 completions (SEGV).

2026-04-18 — live bench pass

  • Clean burn measured: 66 tok/s @64-tok, 33 tok/s @1024-tok post-SEGV fix.
  • Ternary GEMV rocprof: 92% of LPDDR5 peak on the isolated benchmark.
  • sd.cpp native-HIP port promoted to core.

Earlier

Before April 2026 the project was named differently and had a different structure. The monorepo fold happened 2026-04-20 (rocm-cpp/ subtree add). The stampby GitHub handle is retired; canonical is bong-water-water-bong.
