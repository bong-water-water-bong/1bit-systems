Changelog

Canonical source: GitHub releases. v0.1.0 highlights and a few load-bearing cutovers below.

v0.1.0 — first tagged release

  • halo-1bit-2b baseline: 80.8 tok/s @ ctx 64, 68.2 tok/s @ ctx 1024, PPL 9.16 on wikitext-103.
  • Ternary GEMV at 92 % of LPDDR5 peak on the isolated benchmark.
  • Split-KV Flash-Decoding attention default on all context lengths (up to 11.98× @ L=8192, bit-exact).
  • int8 KV default in the router; HALO_KV_DTYPE=f16 override.
  • Cross-arch PPL parity: strixhalo (gfx1151) and RX 9070 XT (gfx1201) bit-exact.
  • OPTC Tier-3 mitigation (amdgpu.dcdebugmask=0x410) green on 2-hour concurrent soak.

2026-04-20 — APU-aggregator pivot

  • Positioning shifts from "ternary runtime for the iGPU" to "APU aggregator for CPU + iGPU + NPU."
  • CPU AVX2 session kernel measured 7261 tok/s-equivalent @ 6912×2048 — CPU lane is a viable prefill surface.
  • Lemonade 10.2 shipped Linux NPU support; VitisAI EP route locked in per Rule E.

2026-04-19 — attention rewrite + RoPE fix

  • Split-KV Flash-Decoding attention kernel landed; 6.78× @ L=2048, bit-exact.
  • RoPE convention fix: interleaved → HF split-half. PPL 524 → ~12 on wikitext; 4.29 → 1.04 on repetition text.
  • KV-cache reset bug fix after ~200 completions (SEGV).
