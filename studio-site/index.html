the APU play

Strix Halo is a CPU + iGPU + NPU on one 128 GB LPDDR5 pool at 256 GB/s. Zero-copy across compute surfaces. No PCIe tax. Most people will benchmark the iGPU against a discrete card and miss the point — an APU that shares one weight buffer between a CPU dispatcher, a tensor iGPU, and an NPU prefiller is the performance king for always-on local inference. We're building for the folks who see that. why →

iGPU — shipping

Radeon 8060S, gfx1151, RDNA 3.5. Native HIP ternary GEMV at 92% of LPDDR5 peak. Our kernels, no hipBLAS.

NPU — deferred

XDNA 2, 50 TOPS. Linux SDK doesn't ship Strix Halo yet. Target: prefill offload (compute-bound, perfect NPU fit). Revisit Q3 2026. status →

CPU — next

Zen5 cores sitting idle while iGPU grinds. Move sampler + dispatcher + tokenizer off the iGPU critical path. Projected 5-15% tok/s without a kernel change.

what this is

1.58-bit ternary

Weights are {-1, 0, +1}. 10× smaller than FP16. The zero doubles as activation sparsity. Microsoft BitNet lineage, our kernels. why →

Rust above, C++ below

HIP kernels in C++20 (rocm-cpp). Everything else — server, router, agents, CLI — in Rust edition 2024. Clean FFI boundary, no interpreters. why →

bare metal, Rule A

No Python at runtime. No containers. No datacenter GPU. Silent-closet inference on a $3k mini-PC, 2.4 MB binary, 200 ms cold start. why →

research lane

activation sparsity

shipped

79.91% measured sparsity in ternary GEMV activations — but weight bandwidth dominates 600:1, so upper bound is 10-15%. Phase 1 done, Phase 2 deferred.

read →

Sherry 1.25-bit

in-flight

3:4 sparsity packing: 2 bits for zero-position + 3 bits for signs. Kernel, requantizer, loader, PPL harness all landed. Bytes-read reduction is rank #1.

read →

BitNet v2 Hadamard

in-flight

Native W1.58A4 via Hadamard rotation (arXiv 2504.18415). Obsoletes the staged a4.8 recipe; evaluating kernel fit for gfx1151.

read →

deferred
