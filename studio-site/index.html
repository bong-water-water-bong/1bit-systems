⚠ CachyOS required

halo-ai targets the AMD Strix Halo iGPU (gfx1151). You need kernel 7.x and the amdxdna NPU driver — CachyOS is the only distro shipping both out of the box today. Ubuntu, Debian, and Arch-base will fail on first run without a manual kernel build + ROCm-7 + NPU backport.

how it stays fast

1.58-bit ternary

Microsoft BitNet weights are {-1, 0, +1}. 10× smaller than FP16. The zero maps to activation sparsity; nothing is wasted. why →

c++ + rust, no interpreters

HIP kernels in C++20. Orchestration in Rust edition 2024. Zero Python at runtime. Zero containers. 2.4 MB binary, 200 ms cold start. why →

bare metal on Strix Halo

$3k mini-PC with 128 GB LPDDR5 shared between CPU + iGPU. No cloud, no PCIe copies, no datacenter GPU tax. Silent-closet inference. why →
