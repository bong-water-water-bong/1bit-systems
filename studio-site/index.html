the stack

rust orchestration

halo-workspace — 11 cargo crates, edition 2024, rust 1.86. Native async/await, tokio, axum. Zero Python at runtime.

amd hip kernels

rocm-cpp ternary GEMV + split-KV Flash-Decoding attention. 92% of LPDDR5 peak on gfx1151. The only public gfx1151 BitNet kernel.

apple mlx path

Feature-gated mlx-apple via bitnet-mlx-rs. Same workspace, Apple Silicon binary. One codebase, two architectures.

openai-compat api

halo-server on :8180. /v1/chat/completions, /v1/completions, /v1/models, /ppl, /metrics. SSE streaming, drop-in for any OpenAI client.

lemonade shim

halo-lemonade on :8200. AMD Lemonade API compat — Sorana, TabNeuron, diffron, any Lemonade-SDK client works without shim code.

self-maintaining

17 halo-agents run in the background — anvil rebuilds kernels, librarian keeps the changelog, quartermaster triages issues, magistrate scans PRs. You work on apps; they keep the stack green.

works with

open webui

Settings → Connections → Add OpenAI API → point at https://strixhalo.local/v2. Full multi-conv, RAG, MCP tools.

librechat

Drop endpoints.custom into librechat.yaml. Runs on any Linux.

dspy (stanford)

dspy.LM("openai/halo-bitnet-1.58", api_base="http://strixhalo.local:8180/v1"). Compile prompt chains against halo-server. Free external validation.

claude code

halo-mcp registers as an MCP server. 17 specialists appear in tools/list. Slash commands + statusline + auto-memory hook via the halo-ai plugin.

halo-gaia

Our own ratatui TUI. Keyboard-only, streaming SSE, works over SSH to a headless box.

bring your own

Anything OpenAI-compat with a base_url setting works. No shim, no token translation, no proprietary protocol.

install

git clone git@github.com:stampby/halo-ai-core.git
cd halo-ai-core
./install-strixhalo.sh

# then:
halo doctor
halo chat
halo bench
halo say "hello strix"

CachyOS recommended — kernel 7.x ships the amdxdna NPU driver + ROCm-7. Other distros need a kernel build first.

The Rust side (halo-ai-rs) is private until launch. Request a read-only collaborator invite on Discord.
