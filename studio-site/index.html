⚠ CachyOS required

halo-ai targets the AMD Strix Halo iGPU (gfx1151). You need kernel 7.x and the amdxdna NPU driver — CachyOS is the only distro shipping both out of the box today. Ubuntu, Debian, and Arch-base will fail on first run without a manual kernel build + ROCm-7 + NPU backport.

install

git clone git@github.com:bong-water-water-bong/halo-ai-core.git
cd halo-ai-core
./install-strixhalo.sh

After ~5 minutes:

halo doctor          # green = you're live
halo chat            # REPL against the local server
halo say "hello"     # TTS through halo-kokoro
halo bench           # parity vs the C++ reference
